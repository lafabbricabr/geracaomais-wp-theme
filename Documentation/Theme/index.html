Fable - Children Kindergarten

Fable - Children Kindergarten WordPress Theme | Created: 23.10.2014 | Latest update: 28.08.2017

About

Fable is a responsive WordPress Theme best suitable for kindergarten, child care center, preschool or nursery, school or education related project. Each and every element of the Theme has been tested to ensure it adapts to modern smarthphones and tablets.

The Theme comes with one click Theme Demo Data Installer so you can import complete set of our Theme Demo pages, posts, widgets with single click. Build-In Drag and Drop Page Builder with extensive collection of components allows to build your pages in minutes. For those who prefer to work with a native WordPress editor there is a build-in Shortcode Generator.

Project structure

Folder name Description
_documentation Documentation of theme.
admin Admin functions.
class All PHP classes used in this theme.
dummy_content Files which contain demo data: content, widget settings.
languages Languages files: .po and .mo.
library PHP libraries used in this project.
media All graphic elements like: icons, navigation arrows etc.
public Public functions.
script ALL JS classes and jQuery plugins.
source ZIP packages of used plugin.
style CSS stylesheets.
template All HTML templates.
